Autoencoders

Autoencoders are a type of ANN used to learn efficient codings of unlabeled data. Autoencoders learn two functions: and encoding function \(h=f(\mathbf{x})\) that transforms the input data, and a decoding function \(r=g(\mathbf{h})\) that recreates the input data from the encoded representation (i.e. it attempts to copy its input to its output). The goal is not for the autoencoder to learn exact mappings (i.e. \(g(f(\mathbf{x}))=\mathbf{x}\)), but to learn an approximate model by prioritising which aspects of the input should be copied and thus (hopefully) learning the most useful properties of the data.
