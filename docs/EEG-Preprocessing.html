EEG-Preprocessing

EEG data preprocessing is generally highly specific to the data and the research project at hand. Much preprocessing will be performed using some form of signal processing, though many options exist. Much data preprocessing is aimed at the removal of signal artifacts

Preprocessing Examples

  • High-pass filter
  • Import channel locations
  • Reference EOG, EKG, EMG
  • Epoch data around important events
  • Subtract pre-stimulus baseline
  • Adjust marker values
  • Manual trial rejection
  • Mark bad electrodes
    • Identify any problematic electrodes that might be introducing artifacts to data.
  • Average reference EEG channels
  • Run ICA to clean data
    • This can be a convenient way to isolate, identify, and remove artifacts in signals.
    • Note that "ideally you should only remove one component from your data, which should be the blink component"
