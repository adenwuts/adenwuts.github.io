Motivations for Rhythm-Based Analyses

"It is well known that the brain exhibits a lot of rhytmic neural activity" -- this applies to rhythms across spatial and time domains.

Phase locked (or evoked) data is synchronized across time, and contrasts with non-phase-locked signals, which are in synchronous. Non-phase-locked signals can cause problems with Time Domain Analyses (ERPs).

Different measurements are possible depending on whether data is time-locked or non-time locked. Note that these terms are only relevant for trial-based experiment design where there is some event that can be referred to as "time 0" (some stimulus that marks the start of a trial, etc).

(Note that time-locked data and phase-locked data are different, see here).
